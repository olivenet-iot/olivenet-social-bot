# Video Prompt Examples

Real prompt patterns for Olivenet content.

## Prompt Structure

### Sora 2 Format

```
[SCENE DESCRIPTION - Detailed scene definition]

Cinematography:
- Camera shot: [wide/medium/close-up]
- Camera movement: [static/dolly/pan/arc/reveal]
- Lens: [35mm/50mm/85mm] + depth of field

Lighting:
- Key light: [main light source and direction]
- Fill: [fill light]
- Mood: [professional/warm/dramatic/calm]

Palette:
- Primary: Olive green #2E7D32
- Secondary: Sky blue #38bdf8
- Accent: [additional color]

Actions:
- [0-2s]: Hook action
- [2-4s]: Main action
- [4-6s]: Closing

Sound:
- Ambient: [environment sound]
- SFX: [effects]
```

### Veo 3 Format (Timestamp-based)

```
[00:00-00:02] [Shot type], [subject], [action]. Ambient: [sound].
[00:02-00:04] [Shot type], [subject], [action]. SFX: [effect].
[00:04-00:06] [Shot type], [final image]. Emotion: [feeling].
```

## Content Templates

### 1. Technology/Product Showcase

```
A sleek IoT sensor device sits on a modern surface. Camera performs slow
180-degree arc around the device. Holographic data visualizations materialize
showing temperature and humidity readings. Clean white background with subtle
olive green accent lighting.

Camera: Medium close-up, slow orbital arc
Lighting: Soft diffused from above-left, blue LED edge glow
Mood: Premium, technological, trustworthy
Palette: White (#FFFFFF), Olive green (#2E7D32), Tech blue (#1976D2)
```

### 2. Greenhouse/Agriculture Scene

```
Morning sunlight streams through greenhouse glass panels. Rows of healthy
green plants with small sensors between them. Camera tracks smoothly along
plant rows, shallow depth of field keeps focus on nearest sensor showing
green status light.

Camera: Wide establishing, then tracking medium shot
Lighting: Natural morning light, soft volumetric rays
Mood: Peaceful, hopeful, natural
Sound: Gentle ventilation hum, distant bird chirps
```

### 3. Data Visualization

```
Dark background with bold 3D animated text forming from digital particles.
Numbers and graphs animate and float in space. Camera slowly rotates around
the composition. Cool blue and green color palette with subtle glow effects.

Camera: Static center, slow rotation
Style: Clean motion graphics, minimal
Palette: Dark (#1a1a1a), Olive (#2E7D32), Blue (#38bdf8)
```

### 4. Before/After Transformation

```
Split screen comparison: Left side shows struggling plants with harsh lighting,
right side shows thriving greenhouse with sensors and soft natural light.
Animated wipe transition moves from left to right, transforming the scene.

Camera: Static center frame
Transition: Horizontal wipe reveal
Contrast: Harsh vs soft lighting, dying vs thriving
```

## Camera Movements

| Movement | Use Case | Effect |
|----------|----------|--------|
| Static | Product-focused | Professional, trustworthy |
| Slow dolly in | Reveal, attention | Curiosity, interest |
| Slow dolly out | Big picture | Context, meaning |
| Pan (horizontal) | Wide scene sweep | Discovery |
| Arc (orbital) | Around product | Premium, 3D effect |
| Reveal | Back to front | Surprise, wow |

## Lighting Setups

### Morning/Natural
- Golden hour quality
- Soft shadows
- Warm tones

### Studio/Product
- Key light: 45 degrees from above
- Fill: 50% intensity
- Rim light: back separation

### Dramatic/Tech
- Single strong source
- Deep shadows
- Blue/green accent

## Brand Colors

| Color | Hex | Usage |
|-------|-----|-------|
| Olive Green | #2E7D32 | Primary brand, nature |
| Sky Blue | #38bdf8 | Technology, data |
| White | #FFFFFF | Clean, modern |
| Dark Gray | #1a1a1a | Background, contrast |

## Guardrails (Avoid)

- Real human faces (quality issues)
- Fast cuts (prefer single scene)
- Text/overlay (add in post-production)
- Complex multi-scene (keep it simple)
- Vague descriptions ("something nice" - be specific)

## Quality Checklist

- [ ] Shot type specified? (wide/medium/close-up)
- [ ] Camera movement defined?
- [ ] Light source and direction included?
- [ ] 3-5 color palette defined?
- [ ] Action beats timed?
- [ ] Sound/ambient defined?
- [ ] 9:16 aspect ratio specified?
- [ ] Duration 5-8 seconds?
