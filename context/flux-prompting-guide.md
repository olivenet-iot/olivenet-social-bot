# FLUX.2 Pro Prompting Guide

## Prompt Framework
**Subject + Action + Style + Context**

Örnek: "Red fox sitting in tall grass, wildlife documentary photography, misty dawn"

## Sıralama Önemli!
En önemli elementler BAŞTA olmalı:
1. Ana konu (Subject)
2. Aksiyon/Poz
3. Stil
4. Bağlam/Ortam
5. İkincil detaylar

## Optimal Uzunluk
- Kısa (10-30 kelime): Hızlı konseptler
- Orta (30-80 kelime): Çoğu proje için ideal ✓
- Uzun (80+ kelime): Karmaşık sahneler

## Olivenet İçin Şablonlar

### Ürün/Teknoloji Fotoğrafı
`[Ürün detayları], [yerleşim], [ışık düzeni], [stil], [mood]`

### Endüstriyel/Tesis
`[Tesis/mekan], [perspektif], [aydınlatma], [stil], [atmosfer]`

### Dashboard/UI Mockup
`[Ekran içeriği], [cihaz], [ortam], [renk paleti], [stil]`

## Renk Kontrolü
FLUX hex kod anlıyor:
- "olive green (#4a7c4a)" ✓
- "sky blue (#38bdf8)" ✓

## Metin Entegrasyonu
- Tırnak işareti kullan: "The text 'OPEN' appears..."
- Yerleşim belirt: "in the bottom left corner"
- Stil tanımla: "bold sans-serif typography"

## Negatif Prompt KULLANMA!
Bunun yerine pozitif alternatifler:
- ❌ "no crowds" → ✅ "peaceful solitude"
- ❌ "without blur" → ✅ "sharp, crisp details"

## Profesyonel Fotoğrafçılık
- Aperture: f/1.4 (bulanık arka plan), f/8 (her yer net)
- Lens: 24mm (geniş), 85mm (portre), 50mm (doğal)
- Işık: "Rembrandt lighting", "golden hour", "soft studio lighting"

## Olivenet Marka Elementleri
- Renkler: olive green (#4a7c4a), sky blue (#38bdf8)
- Ton: Profesyonel, modern, teknolojik
- Tema: IoT, sensörler, data, tarım, enerji
- Atmosfer: Temiz, minimal, güvenilir
